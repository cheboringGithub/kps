# КПС Курс — 30 дней

> Мобильное PWA-приложение для восстановления подвижности крестцово-подвздошного сустава, коррекции перекоса таза и устранения компенсаторной нагрузки на колено.

---

## Проблематика

Программа разработана под конкретный паттерн дисфункции, характерный для людей с сидячей работой:

**Корневая причина — хроническое сжатие КПС (крестцово-подвздошного сустава)**

При длительном сидении подвздошно-поясничная мышца (ППМ) укорачивается и тянет подвздошную кость вперёд и вниз. Ягодичные в ответ перенапрягаются, зажимая крестец с противоположной стороны. КПС оказывается в постоянных «тисках» с двух сторон — отсюда скованность поясницы и ощущение зажатости таза.

**Вторичные проявления:**

- Перекос таза влево (левая седалищная кость давит сильнее при сидении)
- Заторможенность левой средней ягодичной → нестабильность таза в горизонтальной плоскости
- Компенсаторная перегрузка левого колена при ходьбе и беге
- Укорочение правой ППМ → приподнятость правой стороны таза
- Ограниченная подвижность задней поверхности бёдер (невозможность достать ладонями до пола)

## Структура курса

### Фаза 1 · Дни 1–14 · Декомпрессия

Цель — научить КПС расслабляться. Никакой осевой нагрузки на позвоночник, никаких скручиваний. Работа лёжа. Тело учится чувствовать разницу между зажатым и свободным суставом.

Ключевые инструменты:
- Тракция КПС (симметричная и поочерёдная)
- Pelvic Clock — нейромышечная калибровка нейтрали таза
- Фигура-4 — декомпрессия через грушевидную мышцу
- Первичный контакт с ППМ через мягкие выпады

### Фаза 2 · Дни 15–23 · Главные виновники

Работа с подвздошно-поясничной мышцей как основным зажимателем КПС. Параллельно — активация средней ягодичной, которая в норме стабилизирует таз горизонтально, но при перекосе заторможена.

Ключевые инструменты:
- Углублённые выпады с ротацией
- Одноножный мост (приоритет левой ягодичной)
- Поза голубя подготовительная
- 90/90 — наружная ротация тазобедренного сустава
- Первый ПНФ (проприоцептивное нейромышечное облегчение)

### Фаза 3 · Дни 24–29 · Коррекция перекоса

Прицельная работа с асимметрией. ТН-фасция и боковая цепь — ключ к разгрузке левого колена. ПНФ-наклоны приближают ладони к полу как естественное следствие свободного КПС.

Ключевые инструменты:
- ТН-фасция лёжа на боку (левая сторона приоритет)
- Боковой наклон с воздействием на ИТ-тяж
- ПНФ-наклон вперёд
- Полный комплекс активации ягодичных

### Фаза 4 · День 30 · Интеграция

Закрепление нейтрального таза. КПС стабилен и свободен. Программа перехода на поддерживающий режим.

---

## Принципы асимметрии

Программа учитывает конкретную картину перекоса:

| Структура | Состояние | Подход |
|---|---|---|
| Левый КПС | Сжат, опущен | Больше тракции, приоритет в 90/90 |
| Левая средняя ягодичная | Заторможена | Больше повторов на активацию |
| Правая ППМ | Укорочена сильнее | Больше времени в выпаде (правая нога сзади) |
| Правая ягодичная | Перегружена удержанием | Равная работа с левой в фигуре-4 |
| ИТ-тяж слева | Перегружен | Приоритет в боковых растяжках |

---

## Технические особенности приложения

- **PWA** — устанавливается на экран без App Store, работает офлайн
- **Таймер с сигналами** — каждое упражнение имеет структуру подходов (работа / отдых) с звуковыми сигналами через Web Audio API
- **Wake Lock API** — экран не гаснет во время активного таймера
- **Прогресс** — сохраняется в localStorage между сессиями
- **Без верхней части тела** — программа работает только с тазом, бёдрами и нижними конечностями

## Ожидаемые результаты

| Период | Ориентир |
|---|---|
| Дни 1–3 | Субъективное облегчение в пояснице после первых сессий |
| Неделя 2 | Снижение утренней скованности таза |
| Неделя 3–4 | Уменьшение нагрузки на левое колено при ходьбе |
| Неделя 5–6 | Пальцы касаются пола при наклоне |
| Неделя 7–8 | Ладони на полу, таз ощущается симметрично в стойке |

## Поддерживающий режим после курса

4–5 сессий в неделю по 20 минут достаточно для удержания результата. Приоритет: декомпрессия КПС + активация средней ягодичной + одна растяжка задней поверхности.
